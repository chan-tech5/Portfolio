# Chandru S - Personal Portfolio

Welcome to my personal portfolio! This is a modern, high-performance web application built with **Next.js**, **React**, and **Tailwind CSS**. It serves as a digital resume and showcase of my journey, experiences, projects, and skills as an AI Builder, Community Leader, and IT Student.

## 🚀 Live Site
You can view the live portfolio here: **[https://chan-tech5.github.io/Portfolio/](https://chan-tech5.github.io/Portfolio/)**

## ✨ Highlights
- **Premium UI:** Stunning glassmorphism design, floating particles, micro-animations, and dynamic themes.
- **Fully Responsive:** Optimized for all devices, from desktop to mobile.
- **Automated Deployment:** CI/CD pipeline set up to automatically deploy updates.

## 🛠️ Technology Stack
- **Framework:** [Next.js (App Router)](https://nextjs.org/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Animations:** [Framer Motion](https://www.framer.com/motion/)
- **Hosting:** GitHub Pages
